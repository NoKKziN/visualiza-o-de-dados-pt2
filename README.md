## 📊 Sobre o Projeto

Este projeto realiza uma análise exploratória completa do dataset **"Students Performance in Exams"**, disponível no Kaggle. O objetivo principal é responder a perguntas de negócio através de visualizações de dados, aplicando princípios de design estatístico e visual, como os conceitos de Tufte, Gestalt e Cleveland-McGill, para criar gráficos claros, honestos e informativos.

Além da análise exploratória, o projeto também aborda **desafios de refatoração**, onde gráficos com problemas de design são corrigidos para se tornarem mais eficientes e visualmente claros.

## 🎯 Objetivos do Projeto

*   **Explorar** a relação entre variáveis como notas, gênero, raça/etnia, nível de escolaridade dos pais e preparação para o teste.
*   **Responder** perguntas de negócio hipotéticas com visualizações adequadas.
*   **Aplicar** princípios de design de visualização de dados (Data-Ink Ratio, Lie Factor, Chartjunk, Gestalt).
*   **Refatorar** visualizações problemáticas, melhorando sua honestidade e clareza.
